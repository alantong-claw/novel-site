- **Key Learnings & Decisions:**

    - Secure remote voice access is best achieved via `cloudflared` tunnels and restricted firewall rules, rather than exposing services directly to public networks.
    - Formalizing workflows into scripts, documentation, and memory entries is essential for persistence and future reusability.
